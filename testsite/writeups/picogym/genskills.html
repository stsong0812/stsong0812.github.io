Some interesting PicoGym General Skill Challenges.


PW Crack (1-5)

Description: Follows the PW Crack challenges (1-5) from PicoGym. These challenges have to do with "cracking" the password for each level. Each level presents you with a Python-based password checking program as well as an encrypted text file. The goal of these challenges are to analyze and reverse engineer the given Python program in order to decrypt the provided text file.
