Description: Help us test the form by submiting the username as test and password as test! Additional details will be available after launching your challenge instance.

Writeup: The message below mentions redirection, try going back to the login page. Log in with the same credentials once more and observe the redirection in the search bar. It looks like we are initially directed to a different URL before being redirected to the entry page.



Open Burpsuite, we want to capture the request after we log in to further examine the redirection. Observe the GET header which includes the directory we are directed to. The header seems to contain a base64 encoded string, we can decode this string through the command line. Type "echo "base64 string" | base64 --decode". We get one half of the flag.



Forward the request once more, and again, we are given another base64 encoded string in the GET header. Decode this base64 string for the flag.
