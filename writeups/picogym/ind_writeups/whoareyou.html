Who are you?

Description: Let me in. Let me iiiiiiinnnnnnnnnnnnnnnnnnnn http://mercury.picoctf.net:39114/

Writeup: The website displays a message that states only PicoBrowser users are allowed on the site. Intercept the website through Burp and send the request to the repeater.

Change the User-Agent field to "PicoBrowser" and send the new request. In our response, the site now displays the message "I don't trust users visiting from another site."



This time add a Referer header with the same value as the host website to change what host is making the request.



Our next message states, "Sorry, this site only worked in 2018." Add the Date header along with a date that contains the year 2018. Sending our new request, the reponse now states "I don't trust users who can be tracked."



This requires us to include the DNT header to our request, which determines whether or not the user will tracked by websites or not.



This next message requires that our request comes from Sweden. Adding the "X-Forwarded-For" header alters where our request comes from in the form of an IP address. Looking up the range for Swedish IP addresses, we can take any random IP and use it as our X-Forwarded-For value.



Our last message states, "You're in Sweden but you don't speak Swedish?" Requiring us to change the "Accept-Language" field to the Swedish language code in HTTP's syntax.
