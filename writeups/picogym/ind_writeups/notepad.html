Description: Can you read files in the root file? Additional details will be available after launching your challenge instance.

Writeup: List possible commands with sudo -l. We see that we have access to vi (vim), vim is the default editor that comes with UNIX operating systems. We can use vim to execute commands through its command mode.

Open vim with sudo. To run commands through vim preface each command with ":!". Execute :! ls -a /root to list all files, including hidden files, within the root directory. We see the flag file, press enter to execute another command through vim. Next execute :! cat /root/.flag.txt for the flag.
