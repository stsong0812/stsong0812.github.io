Description: What integer does this program print with argument 3297082261? File: chall_2.S Flag format: picoCTF{XXXXXXXX} -> (hex, lowercase, no 0x, and 32 bits. ex. 5614267 would be picoCTF{0055aabb})

Writeup: Open the assembly module in a code editor. With an input argument of 3297082261, we need to determine what the program will return. Our main function prepares the input for the first function in our loop called func1.



This seems to be the starting point for our loop. Observing the code, it looks like a loop occurs until the compare method in the .L2 function returns false.



The program will loop until w1 becomes larger than w0. Knowing the value is incremented by 3 every loop, we can multiply our input number by 3 for our result. Convert our resulting number to hex, and a 32 bit number for our flag.
