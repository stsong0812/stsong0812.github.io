Special

Description: Don't power users get tired of making spelling mistakes in the shell? Not anymore! Enter Special, the Spell Checked Interface for Affecting Linux. Now, every word is properly spelled and capitalized... automatically and behind-the-scenes! Be the first to test Special in beta, and feel free to tell us all about how Special streamlines every development process that you face. When your co-workers see your amazing shell interface, just tell them: That's Special (TM) Start your instance to see connection details.

Writeup: Upon connecting to the server, and attempting to try some standard commands, notice...

  • Every first letter is capitalized ( e.g. clear = Clear)
  • Every word in our command is auto corrected ( e.g. cd = Ad)
We need to utilize commands that are not executed as our first statement, as their first letter will be capitalized. Additionally, our command must be a recognized word to avoid auto-correction.

Using the pipe ("|") operator, we can bypass these filters. The first command before the pipe can be arbitrary, but our second command can be a 'find' command. We can use 'find .' to search through our current working directory. In this search, we identify the 'flag.txt' file located in the '/blargh' directory. Following the same method, this time employ 'cat' to access the contents of the 'flag.txt' file.
