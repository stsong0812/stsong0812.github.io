advanced-potion-making

Description: Ron just found his own copy of advanced potion making, but its been corrupted by some kind of spell. Help him recover it!

Writeup: Download the file, using exiftool to examine the file, the file seems to be corrupted.

Opening the file in hexedit notice the IHDR in the header of our ASCII data. This tells us this file is most likely supposed to be a png file.



Looking at the proper signature of a png file, we can see that our corrupted file has a few discrepancies in its header.



Alter the hex data through hexedit to match the proper png file signature.



Now our file displays as a proper png, but opening the file shows us a completely red image.



We can most likely find our flag within the image through Stegsolve, a tool used to analyze images in different planes by the altering of bits. Scrolling through the different planes, we obtain our flag in red plane 0.
